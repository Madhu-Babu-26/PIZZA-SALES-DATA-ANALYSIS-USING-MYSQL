# Pizza Sales Data Analysis using MySQL

### Project Overview
This project analyzes pizza sales data using MySQL to extract business insights such as total revenue, most popular pizzas, and category-wise performance. It demonstrates SQL skills across basic, intermediate, and advanced query levels.

### Dataset Files
| File Name | Column_Names |
|----------|-------------|
| order_details.csv | order_details_id, order_id, pizza_id, quantity |
| orders.csv | order_id, date, time |
| pizza_types.csv | pizza_type_id, name, category, ingredients  |
| pizzas.csv | pizza_id, pizza_type_id, size, price |
### Key SQL Tasks Performed
- Total number of orders
- Total revenue generated
- Highest priced pizza
- Most common pizza size
- Top 5 ordered pizza types
- Category-wise and hourly order distribution

### Tools & Technologies
- MySQL Server / MySQL Workbench
- SQL (Joins, Group By, Aggregations, Window Functions)
